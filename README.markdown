EngSocTheme
============

Wordpress plugin for all websites hosted by the Skule Wordpress install. Enables modifications to the SkuleBar, as well as other useful features. 

Prerequisites
------------
You should know:

  *   PHP
  *   Wordpress (+ Child Themes)
  *   Buddypress
  *   HTML
  *   CSS
